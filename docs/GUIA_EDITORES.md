# Guia para Editores

## 1. Activar modo editor
1. Ir al panel editorial.
2. Ingresar clave `ADMIN_API_KEY`.
3. Pulsar **Activar modo editor**.

## 2. Revisar propuestas
- En `Propuestas pendientes` revisa contenido y pertinencia.
- Opciones:
  - **Publicar**: cambia estado a `published`.
  - **Archivar**: cambia estado a `archived`.

## 3. Crear categorias
- Usa el bloque `Crear categoria`.
- Define nombre, color, posicion y descripcion.
- Decide si se publica de inmediato.

## 4. Exportar respaldo
- Usa el boton `Exportar JSON`.
- Guarda el archivo generado para backup institucional.

## 5. Buenas practicas editoriales
- Publicar solo con lenguaje claro y aplicable.
- Evitar duplicados semanticos.
- Mantener consistencia entre categoria y etiquetas.
