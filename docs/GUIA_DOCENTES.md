# Guia para Docentes

## 1. Explorar el repositorio
- Usa el buscador para encontrar por palabra clave.
- Filtra por categoria.
- Cambia orden entre `Mas reciente` y `Titulo`.

## 2. Enviar una herramienta
1. Ve a **Enviar nueva herramienta**.
2. Completa categoria, titulo, resumen, tip y conexion PEI.
3. Agrega opciones digitales/analogicas y etiquetas.
4. Envia el formulario.

## 3. Que pasa despues
- Tu propuesta queda en estado `pending`.
- El equipo editorial revisa y decide publicar o archivar.

## 4. Recomendaciones de calidad
- Titulo claro y accionable.
- Resumen con contexto de aula.
- Tip practico aplicable en clase real.
- Conexion PEI explicita.
- Etiquetas concretas (3 a 5).
