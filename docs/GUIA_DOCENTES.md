# Guia para Docentes

## Acceso
- Abre la web publica en GitHub Pages.
- No necesitas login.

## 1. Explorar el repositorio
- Usa el buscador para encontrar por palabra clave.
- Filtra por categoria.
- Ordena por reciente o por titulo.

## 2. Enviar una herramienta
1. Ve a **Aportes de docentes**.
2. Completa categoria, titulo, resumen, tip y conexion PEI.
3. Envia el formulario.

## 3. Como se guarda tu aporte
- Si el backend publico esta activo: se envia al repositorio y queda en `pending`.
- Si el backend no esta activo: se guarda localmente en tu navegador.

## 4. Exportar aportes locales
- Usa **Descargar aportes (.json)** para compartirlos con el equipo editorial.

## 5. Recomendaciones de calidad
- Titulo claro y accionable.
- Resumen con contexto de aula.
- Tip aplicable en clase real.
- Conexion PEI explicita.
- Etiquetas concretas (3 a 5).
